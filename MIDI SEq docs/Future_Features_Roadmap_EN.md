# Future Features & Roadmap — Alpha 1

## Planned Features

### MIDI Clock Synchronization
- MIDI Clock In / Out (planned)
- Ability to act as master or slave in synchronized setups
- Integration with drum machines, sequencers, and DAWs

### Track Control Enhancements
- Solo per track (planned)
- Mute per track (planned)

### Scrub Mode
- Timeline scrubbing for precise navigation
- Non-destructive auditioning of recorded material

### MIDI CC Control
- Extensive control of core device functions via MIDI CC
- External controllers and sequencers can manipulate the device
- Exact CC map to be finalized during development

### Display & UI Expansion
- Planned parallel hardware version with 2.4" SSD1309 display
- Same core firmware philosophy, extended visual feedback

## Development Philosophy
Alpha 1 focuses on validating:
- Core sequencing workflow in real musical use
- Reliability of save/load mechanisms
- Practical user feedback to guide future architectural decisions
