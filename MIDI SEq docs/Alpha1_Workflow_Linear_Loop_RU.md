
# Alpha1 MIDI Sequencer — Workflow (Linear + Loop)

## Общие принципы

Alpha1 — это аппаратный MIDI-секвенсор без встроенного звукового движка.
Для работы **обязательно** требуется:
- MIDI-клавиатура / MIDI-пэды → MIDI IN Alpha1
- Внешний звуковой модуль / синтезатор → MIDI OUT Alpha1

Входящий MIDI:
- записывается в секвенсор
- **одновременно проходит сквозь устройство в реальном времени**, позволяя слышать игру сразу

Alpha1 работает с:
- 16 треками
- 16 MIDI-каналами (Track N = MIDI Channel N)

---

## Linear Recording Workflow (Линейная запись)

### Подготовка

1. Подключите MIDI-контроллер к **MIDI IN**
2. Подключите звуковой модуль к **MIDI OUT**
3. Убедитесь, что звук слышен при игре на клавиатуре
4. Установите:
   - `seq:` → **Linear**
   - BPM (через Encoder → Edit)
   - Текущий трек (Track)

> ⚠️ В Linear режиме запись **деструктивная**  
> Новая запись полностью стирает предыдущие данные текущего трека.

---

### Запись

1. Нажмите **REC**
2. Запись **вооружается**, начинается **Count-in (4 доли)**
   - Пьезо-метроном подаёт клики
   - LED над кнопкой REC мигает в ритм
3. После Count-in:
   - LED над REC **загорается постоянно**
   - Начинается реальная запись MIDI-событий
4. Играйте партию
5. Нажмите **STOP** — запись завершена

> Во время записи слышны все ранее записанные треки + текущий трек в реальном времени.

---

### Запись следующих треков

1. Выберите другой Track:
   - через Encoder (Edit Track)
2. Нажмите **REC**
3. Повторите процесс записи

Максимум:
- 16 треков = 16 MIDI-каналов

---

## Loop Recording Workflow (Запись в петле)

### Подготовка

1. Подключения те же, что и для Linear
2. Установите:
   - `seq:` → **Loop**
   - BPM
   - Track
   - `Loop Length` (длина петли)

> Без задания длины петли Loop-запись невозможна.

---

### Запись в Loop режиме

1. Нажмите **REC**
2. Начинается **Count-in (4 доли)**
3. После Count-in:
   - LED над REC загорается
   - Начинается запись
4. По достижении конца петли:
   - воспроизведение **зацикливается**
   - запись продолжается в режиме **Overdub**

Запись продолжается **бесконечно**, пока не нажата STOP.

---

### Поведение в Loop режиме

- Запись **недеструктивная**
- По умолчанию — overdub
- Нажатие **REC во время записи**:
  → немедленно очищает текущий трек и позволяет записать его заново **без остановки цикла**

---

### Переключение треков во время записи

В Loop режиме **не требуется останавливать запись**:

- **FF** → Track +1
- **REW** → Track -1
- или через Encoder (менее оперативно)

Это позволяет живо записывать несколько треков в одном проходе.

---

### MIDI Panic / Hanging Notes

Из-за зацикливания возможно зависание нот, если длинные ноты пересекают точку wrap.

Решения:
- Избегать записи длинных нот в конце петли
- **FF + REW одновременно** → MIDI Panic (All Notes Off)

---

### Ограничения во время записи

Во время активной записи **заблокированы**:
- Menu
- Save / Load
- Loop Length
- Seq Mode (Linear / Loop)

Это предотвращает случайное разрушение live-перформанса.

---

## Завершение и сохранение

1. Нажмите **STOP**
2. Перейдите в Menu → Save
3. Выберите номер проекта
4. Нажмите Encoder (OK)

Проекты сохраняются во Flash-памяти и сохраняются при отключении питания.

---

## Метроном и индикация

- LED-метроном:
  - активен в Play / Rec
  - выключен в Stop
- Пьезо-метроном:
  - только в Rec
  - первый удар такта выделен более высоким тоном
- Count-in:
  - всегда 4 доли
  - только при записи
