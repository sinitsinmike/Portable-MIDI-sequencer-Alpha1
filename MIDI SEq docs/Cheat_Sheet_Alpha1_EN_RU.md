
Portable MIDI Sequencer – Alpha 1
CHEAT SHEET (EN)

CONTROLS
- Encoder: Navigate / adjust values / press to confirm
- Buttons (8): Transport, Record, Track select, Mode, Save/Load/Clean

MODES
- Linear Mode: Continuous timeline recording
- Loop Mode: Loop-based recording with fixed length

CORE ACTIONS
- Record: Press REC + Play
- Stop: Press Play
- Change Track: Track buttons or encoder
- Change BPM: Tap Tempo or encoder

PROJECT
- Save Project: Save button + project number
- Load Project: Load button + project number
- Clean Project: Clean button + project number

NOTES
- Designed for live MIDI performance recording
- Best used as centerpiece of dawless setup

Портативный MIDI секвенсор – Alpha 1
ШПАРГАЛКА (RU)

ОРГАНЫ УПРАВЛЕНИЯ
- Энкодер: Навигация / изменение значений / подтверждение
- Кнопки (8): Транспорт, запись, выбор трека, режим, сохранение

РЕЖИМЫ
- Linear: Непрерывная запись по таймлайну
- Loop: Зацикленная запись фиксированной длины

ОСНОВНЫЕ ДЕЙСТВИЯ
- Запись: REC + Play
- Стоп: Play
- Смена трека: Кнопки треков или энкодер
- BPM: Tap Tempo или энкодер

ПРОЕКТЫ
- Сохранить: Save + номер проекта
- Загрузить: Load + номер проекта
- Очистить: Clean + номер проекта

ЗАМЕТКИ
- Ориентирован на живое MIDI исполнение
- Центр dawless сетапа
