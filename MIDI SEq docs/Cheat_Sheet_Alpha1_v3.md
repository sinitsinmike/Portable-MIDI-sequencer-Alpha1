
ALPHA1 MIDI SEQUENCER — CHEAT SHEET v3 (FINAL-FINAL)

ENGLISH
------------------------------------------------
PLAY/STOP: Button 1
REC:
- STOP mode: arms recording, waits 4-beat count-in
- LINEAR mode: destructive overwrite
- LOOP mode: overdub
REC during recording: clear current track

TRACK SELECT:
- REC mode: REW = track -1, FF = track +1
- Track number = MIDI channel

ENCODER:
- Rotate: navigate parameters
- Press: enter edit mode (*)
- Rotate: change value
- Press: apply, exit edit mode

BPM:
- Editable only in STOP
- Encoder or TAP TEMPO (Shift)
- * indicates edit state

METRONOME:
- LED: BPM indication (PLAY/REC only)
- Piezo: REC only, first beat accented

COUNT-IN:
- REC mode only
- Fixed 4 beats before recording starts

QUANTIZE:
- STOP mode only
- Post-record
- Per project

PANIC:
- FF + REW: All Notes Off

------------------------------------------------
РУССКИЙ
------------------------------------------------
PLAY/STOP: Кнопка 1
REC:
- В STOP: подготовка записи, ожидание 4-х ударов
- LINEAR: перезапись
- LOOP: overdub
REC во время записи: очистка текущего трека

ВЫБОР ТРЕКА:
- В REC: REW = трек -1, FF = трек +1
- Номер трека = MIDI канал

ЭНКОДЕР:
- Поворот: навигация
- Нажатие: режим редактирования (*)
- Поворот: изменение
- Нажатие: применить

BPM:
- Только в STOP
- Энкодер или TAP TEMPO (Shift)
- * = режим редактирования

МЕТРОНОМ:
- LED: индикация темпа (PLAY/REC)
- Piezo: только REC, первый бит акцентирован

COUNT-IN:
- Только в REC
- 4 удара перед стартом записи

КВАНТИЗАЦИЯ:
- Только STOP
- После записи
- На весь проект

PANIC:
- FF + REW: All Notes Off
